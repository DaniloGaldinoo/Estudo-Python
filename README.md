# Estudo-Python
(PYTHON) - Exercicios e Aulas do Curso em Video.

Este repositório contém exercícios de Python realizados durante estudos, seguindo a organização por tópicos e pastas para facilitar a navegação.

O repositório está organizado em pastas de acordo com os conceitos estudados:

- **Fundamentos**: Exercícios de lógica, operadores, tuplas, listas e manipulação básica de dados.  
- **Estruturas_Repetição**: Exercícios que utilizam loops (`while`, `for`) e controle de fluxo.  
- **Modulos**: Exercícios que utilizam módulos ou bibliotecas externas (como `random` ou `time`).
- **Jogos**: Exercícios que utilizam de mecanicas de games ou parecidos.
